## Project info
- python version: 3.11
- framework: fastapi
- database: sqlite, sqlalchemy, alembic

## Features
- signup
  - get email, first name, last name, privacy & policy accept
  - return verify url
- verify
  - active user
- signin
  - create session
  - return cookie
- get user info
